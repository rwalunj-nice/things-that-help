How It Works

From Symptom to Playbook in Seconds

Describe what you're seeing. The assistant returns ranked root cause hypotheses, investigation steps, and the historical playbook that resolved it before.

🗣️

🧠

Architecture

Solution Architecture

A single FastAPI process serves both the browser chat UI and the Microsoft Teams webhook — no external database, no cloud dependencies.

Chat UI + API
Browser-based chat with session management, message history, end-thread, and feedback modal. All 6 REST endpoints served by FastAPI.
Session Store
In-memory thread-safe dict stores all sessions, message history, and feedback. Dashboard polls /api/threads every 5 s.
Knowledge Engine
Word-boundary keyword scoring over 84 JSON entries. Top-2 matches returned per query. Swap in Bedrock embeddings for production semantic search.

Deployment

Up and Running in 5 Minutes

No cloud account, no API key, no external database. Just Python and a terminal.

1
Required

Clone & Install Dependencies

Navigate to the demo-bot/ directory and install the three Python packages.

# Install fastapi, uvicorn, python-dotenv
pip install -r requirements.txt
2
Required

Configure Environment

Copy the example env file. The HMAC secret is only needed for Teams webhook — leave it blank for local browser testing.

cp .env.example .env
# TEAMS_HMAC_SECRET= ← leave blank for local testing
3
Required

Start the Server

Launch with uvicorn. The server starts on port 8000 with hot-reload enabled.

python -m uvicorn main:app --reload
# INFO: Uvicorn running on http://127.0.0.1:8000
4
Required

Open Browser UI & Dashboard

Open the bot in your browser. Click "Start New Thread", ask a swarm question, then check the live dashboard.

# Chat UI
http://localhost:8000

# Live dashboard (all threads + sentiment)
http://localhost:8000/dashboard
5
Optional — Teams

Expose via ngrok for Microsoft Teams

If you want the bot to respond inside a Teams channel, expose the local server with ngrok and register an Outgoing Webhook.

# Terminal 2 — expose localhost:8000
ngrok http 8000

# Copy the https URL → Teams channel → ··· → Manage channel
# → Outgoing Webhooks → Add → Callback URL → paste ngrok URL/webhook
# → Copy HMAC token → paste into .env as TEAMS_HMAC_SECRET → restart
6
Optional — Teams

Test in Teams Channel

Mention the webhook name followed by any swarm symptom. The bot replies inline within 3 seconds. Teams sessions also appear on the dashboard.

@Swarm Assistant agents cannot log in
@Swarm Assistant recordings stuck pending upload
@Swarm Assistant playback error 8203 CXone recording
